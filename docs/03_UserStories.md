# User Stories

## Introduction

This document describes the functional requirements of the HR Platform from the user's perspective.

Each user story follows the standard Agile format:

> As a <role>, I want <goal>, so that <benefit>.

---

# Module 1 - Authentication

## Employee

- As an employee, I want to log into the platform securely so that I can access my personal dashboard.

- As an employee, I want to reset my password so that I can recover access to my account.

---

## Manager

- As a manager, I want to securely authenticate so that I can manage my team.

---

## HR

- As an HR administrator, I want to create employee accounts so that new employees can access the platform.

- As an HR administrator, I want to deactivate user accounts so that former employees cannot access company data.

---

# Module 2 - Employee Profile

## Employee

- As an employee, I want to view my personal information.

- As an employee, I want to update my phone number and address.

- As an employee, I want to upload my profile picture.

---

## HR

- As HR, I want to edit employee information when necessary.

---

# Module 3 - Leave Management

## Employee

- As an employee, I want to submit a leave request.

- As an employee, I want to see the status of my leave requests.

- As an employee, I want to cancel a pending leave request.

---

## Manager

- As a manager, I want to approve leave requests.

- As a manager, I want to reject leave requests.

- As a manager, I want to view my team's leave calendar.

---

## HR

- As HR, I want to configure leave policies.

- As HR, I want to monitor all leave requests.

---

# Module 4 - Documents

## Employee

- As an employee, I want to download my employment contract.

- As an employee, I want to access my payslips.

---

## HR

- As HR, I want to upload HR documents.

- As HR, I want to organize documents by employee.

---

# Module 5 - Onboarding

## Employee

- As a new employee, I want to complete my onboarding checklist.

- As a new employee, I want to know my first-day tasks.

---

## Manager

- As a manager, I want to monitor onboarding progress.

---

## HR

- As HR, I want to assign onboarding tasks automatically.

---

# Module 6 - Performance Evaluation

## Employee

- As an employee, I want to view my objectives.

- As an employee, I want to consult my evaluations.

---

## Manager

- As a manager, I want to evaluate my employees.

- As a manager, I want to assign objectives.

---

## HR

- As HR, I want to monitor evaluation campaigns.

---

# Module 7 - Notifications

## Employee

- As an employee, I want to receive notifications when my leave request is approved.

---

## Manager

- As a manager, I want to receive notifications when a new request requires my approval.

---

## HR

- As HR, I want to notify employees about important HR announcements.

---

# Module 8 - AI Assistant

## Employee

- As an employee, I want to ask HR questions to an AI assistant.

- As an employee, I want AI to explain company policies.

---

## Manager

- As a manager, I want AI to summarize employee performance.

- As a manager, I want AI to suggest meeting summaries.

---

## HR

- As HR, I want AI to summarize employee files.

- As HR, I want AI to classify HR documents automatically.

- As HR, I want AI to generate onboarding plans.

- As HR, I want AI to answer repetitive HR questions.

---

# Future User Stories

Additional user stories will be added as the platform evolves.
