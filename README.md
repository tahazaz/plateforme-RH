# Plateforme RH

## Description

La Plateforme RH est une application web permettant de centraliser les processus liés aux ressources humaines au sein d'une entreprise.

Elle est composée de trois espaces distincts :

- Employé
- Manager
- Ressources Humaines (RH)

Chaque utilisateur dispose d'un tableau de bord adapté à son rôle afin de simplifier la gestion des activités RH quotidiennes.

---

## Objectifs

La plateforme a pour objectif de :

- Centraliser toutes les informations RH.
- Simplifier les échanges entre les employés, les managers et le service RH.
- Automatiser les processus administratifs.
- Intégrer des fonctionnalités d'Intelligence Artificielle pour assister les utilisateurs.
- Offrir une interface moderne, sécurisée et intuitive.

---

## Modules principaux

- Authentification
- Gestion des utilisateurs
- Gestion des rôles
- Gestion des employés
- Gestion des congés
- Gestion des documents RH
- Onboarding
- Evaluations
- Notifications
- Dashboard
- Assistant IA

---

## Les rôles

### Employé

- Consulter ses informations
- Demander un congé
- Télécharger ses documents
- Consulter le Handbook
- Suivre son onboarding

### Manager

- Gérer son équipe
- Valider les congés
- Réaliser les évaluations
- Suivre les objectifs

### RH

- Administrer la plateforme
- Gérer les employés
- Gérer les contrats
- Gérer les documents
- Configurer les règles RH
